== Description ==

Met de klantbeoordeling plugin voor WordPress kunt u de klantbeoordelingen tool van WStools.nl gemakkelijk integreren in uw WordPress website.
Voor deze plugin is een geregistreerde account nodig van [WStools.nl](http://wstools.nl/).

= Zoekmachine geoptimaliseerd =
Wanneer u genoeg ervaringen heeft ontvangen worden de klantbeoordelingen ook in de zoekmachine getoond met sterren. Zowel bij uw home pagina als in uw advertenties. De kans dat mensen klikken op uw advertentie wordt hierdoor vergroot.

= Zelf vragen instellen =
Naast een algemene beoordeling kunt u de bezoeker zelf een beoordeling vragen. U kunt hierdoor bijvoorbeeld vragen wat bezoeker vond van de service en/of de prijs.

= Beheer zelf uw klantbeoordelingen =
U kunt zelf in ons systeem uw beoordelingen beheren. Geef bijvoorbeeld als website eigenaar een reactie op een beoordeling.

== Installation ==
Na het activeren van de plugin kan er een widget geplaatst worden in uw sidebar. Om de klantbeoordelingen te tonen in uw teksten kunt u [WStools-klantbeoordelingen] gebruiken als shortcode.
